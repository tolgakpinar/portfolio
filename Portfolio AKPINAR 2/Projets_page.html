Portfolio Wordpress





Voici un projet scolaire qui était de déployer un portfolio en ligne.









C'était un projet de groupe qui consistait a créer et héberger un portfolio en se servant de WordPress comme CMS (Content Management System ou système de gestion de contenu). Un CMS est un logiciel en ligne grâce auquel il est possible de créer, de gérer et de modifier facilement un site web, sans avoir besoin de connaissances techniques en langage informatique. Notre groupe était compose 4 membres dont 2 avaient le role de développeurs, un membre pour le role de designer web et un autre pour celui de Web Master.





Ressources Technologiques





WordPress : CMS. IONOS : hébergement web et nom de domaine. Figma : logiciel de désigne et de conception. FileZilla : client ftp.

Voici Gestion de Retard Ecole Iris (GREI) Web.









Objectifs





Ce projet sert a gérer les données de l'école et a prévoir les retards des étudiants grâce à l'API (application programming interface ou interface de programmation d'application) d'Île-de-France Mobilités qui nous permet de vérifier les lignes de transports perturbées et prévoir des billets pour les étudiants ayant cours à ce moment précis. Le site est hébergé sur un serveur LAMP qui récupère les données de l'API chaque 5 min grâce à une tache programmé sur la cruntab.





Ressources Technologiques






Apache : serveur web.
MySQL : SGBDR pour gérer la base de données.
PHP : Language de programmation.
Visual Studio Code
